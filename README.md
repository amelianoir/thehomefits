# The Home Fits 🏠✨

Dopamine-style home improvement blog with real-time Amazon affiliate products.

## Features
- 6 rooms with 10 improvement tips each
- 5-6 Amazon products per tip
- Real-time product data via Product Advertising API
- Bright, vibrant dopamine design
- Animated hero section
- Mobile responsive

## Rooms
- Bathroom
- Bedroom
- Living Room
- Kitchen
- Home Office
- Entryway

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- API: Amazon Product Advertising API
- Hosting: Node.js compatible hosting

## Setup
1. Clone repo
2. Install dependencies: `npm install`
3. Create `.env` file with AWS credentials
4. Run: `npm run dev`
5. Build: `npm run build`

## Environment Variables
```
REACT_APP_AWS_ACCESS_KEY=your_access_key
REACT_APP_AWS_SECRET_KEY=your_secret_key
REACT_APP_PARTNER_TAG=thehomefits-20
REACT_APP_AWS_REGION=us-east-1
```
